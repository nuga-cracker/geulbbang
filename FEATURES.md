# 글빵(GeulBbang) 기능 정리

> 마지막 업데이트: 2026-07-07

글빵은 “초고 = 반죽 → 첨삭 = 반죽 치대기 → 고쳐쓰기 = 빵 굽기” 흐름으로 글쓰기 실력을 키우는 브라우저 기반 글쓰기 도구입니다. 서버, DB, 외부 API 없이 정적 웹앱으로 동작하며, 사용자의 글과 진행 상태는 브라우저 `localStorage`에 저장됩니다.

---

## UI 컨셉 — 1인칭 제빵소 작업대 타이쿤

글빵은 단순한 글쓰기 앱이 아닌, **작은 마을 제빵소 주방에 들어와 작업대 앞에서 글 반죽을 만드는** 1인칭 타이쿤 스타일 UI를 제공합니다.

```text
사용자가 작은 마을 빵집 주방에 들어와,
나무 작업대 앞에서 오늘의 주문서를 받고,
글 반죽을 노트에 적고,
오븐에 넣어 다정한 첨삭 메모를 받는
포근한 1인칭 제빵소 타이쿤 화면.
```

### 주요 UI 컨셉 매핑

| 기능 요소 | 제빵소 컨셉 |
|---|---|
| 글감 카드 | 📜 오늘 들어온 주문서 |
| 글쓰기 칸 (textarea) | ✍️ 반죽 노트 |
| 빵 굽기 버튼 | 🔥 오븐에 넣고 첨삭받기 |
| 첨삭 결과 | 👨‍🍳 제빵사의 메모 |
| 비문 다듬기 | 🧁 반죽 손질 제안 |
| 진열장 | 🗂️ 빵 진열 선반 |
| 레벨 표시 | 견습 제빵사 Lv.X |

### 색감 / 디자인 톤

## 0. 글감 데이터베이스 (201개 이상)

매일 글쓰기 시작이 막막하지 않도록 다양한 글감 데이터베이스를 제공합니다.

### 글감 구조

각 글감은 아래 정보를 포함합니다.

| 항목 | 내용 |
|---|---|
| 카테고리 | 글의 주제 분류 |
| 난이도 | 쉬움 / 보통 / 어려움 |
| 주제 | 실제 글감 문장 |
| 목표 | 이번 글에서 도전할 쓰기 목표 |
| 첫 문장 힌트 | 시작이 막힐 때 참고할 첫 문장 |

### 카테고리 구성

- 일기
- 감정
- 상상
- 학교/일상
- 음식
- 관계
- 여행
- 미래
- 자기소개
- 논리/주장
- 창작
- 관찰
- 계절/날씨
- 감사
- 회고
- 취향
- 사회/기술
- 짧은 에세이

### 글감 사용 버튼 (제빵소 UI)

- **이 주문으로 시작**: 현재 표시된 주문서(글감)의 주제와 첫 문장 힌트를 반죽 노트(글쓰기 칸)에 삽입합니다.
- **다른 주문 보기 🎲**: 랜덤으로 다른 주문서(글감)를 표시합니다. 반죽 노트는 자동으로 바뀌지 않습니다.
- **3문장 반죽 틀 넣기 ✏️**: 주제와 3문장 형식의 작성 틀을 반죽 노트에 넣습니다.

### 날짜 기반 오늘의 글감

- 날짜에 따라 매일 다른 글감이 기본으로 표시됩니다.
- `다른 글감 보기` 버튼으로 언제든 다른 글감으로 변경할 수 있습니다.

## 1. 기본 글쓰기 첨삭 게임

### 글쓰기 → 빵 굽기 흐름
- 사용자가 글쓰기 칸에 글을 입력합니다.
- `빵 굽기` 버튼을 누르면 규칙 기반 분석이 실행됩니다.
- 분석 결과로 점수, 피드백, 격려 문구가 표시됩니다.
- 사용자는 피드백을 보고 글을 다시 고칠 수 있습니다.

### 규칙 기반 첨삭 항목
- 긴 문장 감지
- 반복 단어 감지
- 불필요한 군더더기 표현 감지
- 수동태/부자연스러운 표현 패턴 감지
- 존댓말/반말 섞임 감지
- 문장 통계 기반 점수 산정

### 성장 요소
- 글을 굽고 고치면 EXP를 얻습니다.
- EXP에 따라 레벨이 오릅니다.
- 반죽, 식빵, 모닝빵, 크루아상, 바게트 등 빵 등급으로 성장을 표현합니다.
- 반복 수정할수록 보너스 EXP를 받을 수 있습니다.

## 2. 작가 문체 레시피 시스템

사용자가 원하는 문체 레시피를 선택하면 기본 첨삭에 더해 해당 스타일에 맞는 문체 조언이 표시됩니다.

### 제공 레시피
- 🌫️ 부끄러운 고백빵
- 🌸 노래하는 진달래빵
- 😄 구수한 시골빵
- 😏 재치있는 티타임빵
- 🌸 매일 다정 일기빵
- 🕯️ 차분 사색빵

### 주요 기능
- 레시피 카드 선택 UI
- 선택한 문체에 맞는 격려/수정 방향 제공
- 진열장에 저장된 글에 사용한 레시피 태그 표시
- 레시피 사용 기록 기반 배지 획득

## 3. 나만의 시그니처 문체 만들기

사용자가 직접 자신의 글쓰기 취향을 반영한 “나만의 시그니처 빵” 레시피를 만들 수 있습니다.

### 문체 생성 방식
1. **설문 기반 설정**
   - 글 분위기
   - 문장 리듬
   - 말투
   - 선호 단어

2. **내 글 분석 기반 문체 지문**
   - 평균 문장 길이
   - 자주 쓰는 단어
   - 질문/감탄 사용 경향
   - 존댓말/반말 경향

3. **슬라이더 미세 조정**
   - 문장 길이
   - 표현 밀도
   - 에너지
   - 유머감

### 연동 기능
- `나만의 시그니처 빵`을 일반 레시피처럼 선택 가능
- 첨삭 결과에 시그니처 문체 조언 반영
- 진열장 저장 시 시그니처 레시피 태그 표시
- 관련 배지 획득 가능

## 8. 비문/어색한 문장 다듬기

기존 첨삭에 더해 한국어 비문이나 어색한 표현을 부드럽게 제안하는 기능을 추가했습니다.

### 감지 예시
- 조사 중복
  - `나는은`
  - `학교에에`
  - `책을를`

- 연결어 반복
  - `그리고 그리고`
  - `그래서 그래서`
  - `하지만 하지만`

- 종결 표현 반복
  - `했다 했다`
  - `입니다 입니다`
  - `같다 같다`

- 이유 표현 중복
  - `때문입니다 때문이다`

- 장황한 표현
  - `하는 것이 좋을 것 같다`

- 번역투 표현
  - `에 있어서`
  - `에 대하여`
  - `을 통해서`

### 표시 방식
- 기존 첨삭과 별도로 `🧁 비문 다듬기` 섹션 표시
- 원문 일부, 문제 설명, 수정 방향 제공
- 규칙 기반 분석이므로 “비문일 수 있어요”, “더 자연스럽게 바꿔볼 수 있어요”처럼 부드러운 말투 사용

## 9. 제안 바로 적용하기

비문 다듬기 중 자동 수정이 안전한 항목에는 `이 제안 적용하기` 버튼을 제공합니다.

### 동작 방식
1. 사용자가 글을 입력합니다.
2. `빵 굽기`를 누릅니다.
3. `🧁 비문 다듬기` 섹션에 제안이 표시됩니다.
4. 자동 치환 가능한 항목에는 `이 제안 적용하기` 버튼이 표시됩니다.
5. 버튼을 누르면 글쓰기 칸의 문장이 실제로 수정됩니다.
6. 버튼은 `✅ 적용 완료` 상태로 바뀝니다.
7. 사용자는 다시 `빵 굽기`를 눌러 개선 결과를 확인할 수 있습니다.

### 자동 적용 예시
- `나는은` → `나는`
- `책을를` → `책을`
- `학교에에` → `학교에`
- `친구가가` → `친구가`
- `그리고 그리고` → `그리고`
- `했다 했다` → `했다`
- `입니다 입니다` → `입니다`

### 자동 적용하지 않는 항목
문맥에 따라 수정 방식이 달라질 수 있는 표현은 자동 적용하지 않고 안내만 제공합니다.

예:
- `에 있어서`
- `에 대하여`
- `을 통해서`
- 장황한 표현
- 명사화 표현
