# 🍞 글빵 (GeulBbang)

> 맛있는 글을 만들어주는 제빵소 — 글쓰기 첨삭 + 성장형 게임

투박한 초고(반죽)를 첨삭 피드백에 따라 여러 번 고쳐 쓰면서, 점점 더 좋은 빵(유려한 글)으로 구워내는 성장 게임입니다. 🥖

## 🍞 컨셉

| 제빵 과정 | 글쓰기 과정 |
|-----------|-------------|
| 밀가루/반죽 | 처음 쓴 초고 |
| 반죽 치대기 | 첨삭(선생님 피드백) 받기 |
| 발효 | 피드백을 반영해 다시 쓰기 |
| 굽기 | 여러 번 고쳐서 문장이 익어감 |
| 완성된 빵 | 잘 다듬어진 유려한 글 |

## ✨ 특징

- 💸 **유지비 0원** — 서버·API·데이터베이스 없이 브라우저에서만 동작
- 🖋️ **선생님 말투 첨삭** — 규칙 기반, 다정한 피드백 (API 키 불필요)
- 🎮 **성장 게임** — 경험치·레벨·빵 등급·배지·진열장·오늘의 빵
- 💾 **자동 저장** — localStorage로 진행 상황 유지

> 🚧 현재 초기 구현 준비 중입니다. 곧 전체 기능이 추가됩니다!
